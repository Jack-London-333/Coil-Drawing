## 本版修复内容（针对 v202607132026 用户问题）

上一版把整个鼻端 U 形放在电机端面（XY 平面）内、多匝沿轴向叠成“墙”，
导致轴向看是“凸”形、切向看是“小船”。本版按用户确认的真实形态重定向：

- 每个 nose 的 180° 圆冠（中心线半径 `Rc = RD + WA/2`）位于**包含电机
  轴向的竖立平面**内；该平面的法向（即专利定义的鼻端中心线、鼻端芯轴
  方向）落在端面内、与径向直径成 `seita3` 角（`seita3=90°` 时为纯切向）。
- 圆环直径方向的径向分量为 `sin(seita3)`，从上层臂端桥接到半径更大的
  下层臂端。
- 四条等长直鼻臂 `Larm = (LLM - L0)/4` 沿电机轴向朝端外，Q→P 直臂与
  圆冠 G1 相切；闭合中心线总长严格等于专利 `LLM`。
- 多匝在鼻端沿圆环法向按 `HBD` 节距嵌套，如同各匝套在同一鼻端芯轴上；
  接线侧换匝 smoothstep 沿该法向偏移一个 `HBD`，出线侧无独立台阶。
- 匝序拓扑保持不变：下层靠槽口半匝 + 上层靠槽底半匝 = 原始梭形线圈
  最内侧匝；下层靠槽底半匝 + 上层靠槽口半匝 = 最外侧匝。

### 问题一

如图![](C:\Users\19144\Desktop\CoilDrawing\some_user_problems\software-v202607132348-test\pics\轴向看铜导线 宽边朝着用户 这不对.png)

所示，当用户从轴向看铜导线，Nose的 宽边朝着用户，这不对，而应该是窄边朝着用户。

再向你说一个“细节”，你把这一把线圈放入槽中后，你从端部垂直于轴线作平面，以电机轴线为圆心不断地从小到大作圆（半径为r），这样就可以和这一把线圈的“窄矩形”截面相交，得到一个“弧线段”，这个“弧线段”要小于 2×Pi×r÷108 。所以从这里也可以得出，从槽口到鼻部的扭转段，也要尽可能让“矩形的窄边”朝向轴线。

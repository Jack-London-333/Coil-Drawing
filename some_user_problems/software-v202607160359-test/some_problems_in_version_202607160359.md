## 本版修复内容（针对 v202607142334 用户问题：卷环盘姿态/seita3 失效/交叉干涉）

保留 v202607142334 已确认正确的鼻端拓扑（同心嵌套
`R_i = Rc + HBD·(i-(N-1)/2)`、两臂交叉后卷回扫角 230°、同心卷回
天然翻面零扭转、LLM 总长守恒、`Rc=RD+WA/2`），只重构卷环的
**姿态**：

1. **盘面贴合带面（0 偏航）**：删除旧版为撑开交叉净距而引入的
   "环平面绕鼻端中心线偏航 yaw≈30°"——那就是"厚硬币/蜗牛壳斜骑
   在交叉点上"的根源。现在环平面严格由切向和鼻端中心线张成，
   即鼻端处两臂的带面：整个卷环是带子在自己所在的面内卷出来的，
   径向看是正圆的同心卷环盘，不再凸出带面。
2. **seita3 直接控制鼻端姿态**：鼻端中心线（环顶点方向）与径向
   直径严格成 seita3 角（专利定义），盘面随之整体旋转。用户改
   seita3（常用 70–90°），鼻端倾角立即变化（见 70°/80°/90° 三张
   鼻端特写）。LLM 总长差值只由环位 shift/斜边长度吸收，姿态不再
   参与凑长。
3. **交叉错距 = 浅螺旋**：交叉处两臂所需的错距不再靠拧转盘面，
   而是沿整个卷环以 smoothstep 剖面均匀消化（锁紧垫圈式浅螺旋，
   两端斜率为零，与 rd2 圆角严格 G1 对接；鼻尖严格落在盘面内的
   鼻端中心线上）。错距 drop 由**逐匝纤维净距**反解为恰好无干涉
   的最小值——旧版按束中心线净距求解，覆盖不了外匝 rd2 弯角
   纤维斜穿对臂的情形，这正是 v202607142334"已知问题 1"
   （N=8 残余干涉 0.6 cm³）的根源。
4. **臂-环连接平滑**：两臂→rd2 圆角→卷环全链 G1 相切（rd2 圆角
   仍通过有界方程精确求解、切点恰为环端 Q），带宽面连续潜入
   盘缘，无皱折急转。

实现要点：

- 卷环几何由 `_NoseCurl`（平面圆 + 环法向 smoothstep 高度剖面）
  统一描述；高度剖面是三次多项式，束中心线的 STEP 边用柱面上的
  **精确三次 Bézier p-curve** 表示（几何零误差），弧长用 32 点
  Gauss-Legendre 积分（误差 ≪1e-12 mm），闭合中心线总长严格等于
  专利 LLM。
- 卷环实体（整环与换匝螺旋统一）为密站位直纹放样（站位角
  ≤4°），匝绝缘按"主体段 + 每卷环四条无孔条带"拆件，任意匝数
  仍全部是单一闭合实体叶节点。

## 已知事项（如实记录）

1. 浅螺旋错距 `drop≈28.6 mm`（N=8）由零干涉硬约束决定：交叉处
   两臂是 25.5 mm 高的整束slab，加上外匝 rd2 弯角纤维的扫掠范围，
   物理上就需要这么大的让位。若希望观感更扁，可讨论减小交叉半角
   β 或压缩净距裕量（当前每侧 1.5 mm）。
2. 铜引线与防晕层的贴身穿越（约 25 mm³/侧）为历史遗留
   （v202607140123 起即存在），与本次改动无关，可另行开孔处理。

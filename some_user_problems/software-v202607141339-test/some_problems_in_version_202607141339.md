## 本版修复内容（针对 v202607140123 问题一）

上一版把槽侧姿态到圆冠姿态的截面扭转放在 Q→P 直鼻臂内，红箭头处
（rd2 弯角与圆环之间）出现拧转。本版按用户确认的方案把扭转段移出
鼻部：

- **鼻部（rd2 弯角 → Q→P 直鼻臂 → 圆冠）完全零扭转**：圆冠姿态
  （截面薄方向 = 鼻端芯轴方向，HBD 嵌套）沿直臂和 rd2 弯角刚性
  延伸，鼻部没有任何拧转。切向看鼻部：圆环与两条短臂干净利落，
  宽面正对观察者。
- **扭转段位于端部斜边贴近鼻部的一段**（占斜边长度的 35%，
  smoothstep 分布，两端速率为零、与相邻刚性段 G1 对接）：斜边靠
  槽口的 65% 保持槽内姿态（窄边朝向轴线），临进 rd2 前平滑扭到
  圆冠姿态——与真实线圈"临近鼻部集中扭转"的形态一致。
- 圆冠姿态的法向符号按扭转角最小选取（四条斜边扭转约
  ±67°/±87°）。
- 防晕层沿斜边的延伸段改为按同一扭转分布律采样截面，与导线逐点
  一致。
- 圆环嵌套、`Rc = RD + WA/2`、HBD 芯轴节距、LLM 守恒、匝序拓扑
  均不变。

## 几何约束说明

槽内截面薄边必须朝径向（匝沿槽深叠），鼻端圆环截面薄边必须朝
鼻端芯轴方向（否则相邻匝圆环沿芯轴占位 9.3 mm > HBD 3.65 mm 而
互相穿体，也与专利 `Rc=RD+WA/2` 矛盾），两端姿态相差约 67°——
这段扭转在"槽口→鼻部"路径上必须存在。本版将其压缩在斜边贴近
鼻部的区段内，鼻部本体保持零扭转。

## 问题一

![](pics\启用3D Interconnect.png)如第一张图所示，当我使用 SolidWorks 打开导出的3D模型“coil_3d.step”时，如果在“设置-系统选项-导入”中选择“文件格式：普通”，并勾选“启用3D Interconnect”，那么"coil_3d.step"就会作为Assembly中的一级子装配体，并具有二级子零件。此时，这些二级子零件的中文名称都是对的，但是3D模型视图中的零件全是白颜色。


![](C:\Users\19144\Desktop\CoilDrawing\some_user_problems\software-v202607111747-test\pics\不启用3D Interconnect.png)如第二张图所示，SolidWorks其他打开条件不变，如果不勾选“启用3D Interconnect”，那么这些子零件都变成一级子零件了。此时，这些二级子零件的中文名称都显示不出来了，而3D模型视图中的零件颜色都是对的、都是上色过的。此外，导线 和 匝间云母带 会出现“警告：发生重建模型的错误”。

## 本版修复内容（针对 v202607132348 问题一）

上一版把槽侧姿态到鼻端姿态的截面扭转分布在整条端部斜边上、绕斜边
弦线旋转，中途导线宽面完全朝向轴向观察者（宽面法向轴向分量最大约
0.92），宽边也横到了圆周方向。本版按真实线圈重新分配扭转：

- **端部斜边零扭转**：截面保持槽口弯角传递的槽内姿态。斜边上与
  弦线垂直且保持水平的材料薄方向恰好就是槽侧姿态（与 rd2 弯角轴
  只差约 0.8°），因此窄边全程朝向轴向观察者，宽面基本位于径向-
  轴向平面内。
- **rd2 弯角是平弯**：弯曲轴≈截面薄方向，截面姿态刚性通过，不再
  需要预扭。
- **扭转集中在 Q→P 直鼻臂**：绕竖直切线做 smoothstep 扭转（两端
  速率为零，与弯角/圆冠 G1 对接），把槽侧姿态转到圆冠姿态。绕竖直
  轴扭转时材料薄方向始终水平——窄边全程朝向轴向观察者。圆环法向
  符号按最小扭转选取，四条直臂扭转约 ±67°/±87°（此前方案需要
  93°–113° 且中途宽面朝上）。这正是真实线圈鼻部“套芯轴涨形”时
  紧贴鼻端的集中扭转段。
- **圆冠保持不变**：仍是套在鼻端芯轴（法向与径向直径成 seita3）上
  的 180° 圆环，匝间沿芯轴按 HBD 嵌套，`Rc = RD + WA/2`，闭合中心
  线总长严格等于专利 `LLM`。
- 换匝放样与外围方壳的直鼻臂段改为**同站位直纹扭转墙**，逐点嵌套，
  零间隙不穿体；圆冠段仍为光顺放样。

# CoilDrawing — 电机定子成型线圈计算与三维建模

交流电机定子**成型线圈（梭形/菱形硬绕组线圈）**参数计算、大样图绘制与三维建模的小工具，
公式体系来自专利 **CN104965948B《一种交流电机定子绕组线圈参数的计算方法》**（湘潭电机，2015），
即业内常见的"线圈参数计算"内部软件（如 sf.exe）所对应的技术路线。

## 功能

- **计算引擎**（`coildrawing/engine.py`）：专利步骤 1–6 全流程
  1. 导线绝缘后尺寸（支持两种线规并绕）
  2. 槽内/端部截面尺寸，槽内剩余空间验证
  3. 线圈空间几何模型（RR1/RR2/张角 fai/齿距 t1,t2/鼻端投影 E1,E2）
  4. 上下层边弯弧判定（D 与 RR1 比较，弧长/弦长选择）
  5. 端部迭代计算：轴向投影长 CC、端部斜边弧长 S1/S2、直线部 X1/X2、
     弯弧系数 K1/K2、**平均匝长 LLM**、单线圈用线长
  6. **梯形梭形参数**（绕线模大样：L4/L5/XX1/h_/梭长 Lm1）
- **桌面 UI**（PySide6）：参数输入（自由文本数值框，支持 8 位小数，
  滚轮不改值）、结果表（8 位小数显示）、大样图预览、一键导出
- **工作空间**：每次正常启动都先弹出目录选择窗口，默认定位到上次使用的目录；
  用户可以确认该目录或浏览选择其他目录，取消则直接退出软件且不写入任何目录。
  确认空目录后会自动初始化 **config.txt** 与 `output\`。参数文件为带中文注释的
  INI 风格纯文本，可手工编辑并由软件检测外部修改；也可导入任意名字的配置 txt
  （复制改名为 config.txt）
- **大样图**（matplotlib，工程制图风格）：端面投影图 + 端部侧视图 +
  **槽内截面图（逐匝真实截面）** + 梯形梭形大样，带图框/标题栏，
  可导出 PNG / PDF / SVG（矢量）
- **数据导出**：Excel（输入/结果/警告分表）、CSV
- **三维模型**（build123d/OCCT），导出 STEP 装配体（部件带中文名称与颜色，
  名称按 ISO 10303-21 `\X2\` 转义写入，SolidWorks 等可正确显示、可改名、可进 BOM；
  所有部件均为**有效封闭实体**；导出前逐件检查，写出 STEP 后再次回读检查，
  Compound、开放壳或曲面模型会被拒绝而不会静默导出）：
  - **逐匝精细模型**：一根连续扁铜线按真实绕制拓扑形成 N 匝
    （并绕时每股各自连续，支持 NPD、NCD 与双线规）。若涨形前由内到外
    编为 `1…N`，涨形后上层槽口→槽底为 `N…1`，下层槽口→槽底为
    `1…N`；两根引线都靠槽底。两个 nose 是**交叉卷环**：两条端部
    斜边收拢、交叉，环从交叉点外侧卷回，扫角 `180°+2β`（β=25°）；
    `RD` 是内弯半径，束中心环半径 `Rc=RD+WA/2`。多匝在环平面内
    **同心嵌套**（半径差=端部每匝高度 HBD，内匝小外匝大），同心
    卷回天然完成匝序翻面，鼻部零扭转。环平面就是鼻端处两臂的带面
    （0 偏航）：由切向和鼻端中心线张成——`seita3`（度）按专利表示
    鼻端中心线与径向直径的夹角（常用 70°–90°），直接控制鼻端姿态；
    环随 F 整体向槽底抬高。交叉处两臂所需错距沿整个卷环以
    smoothstep 浅螺旋均匀消化（升角仅几度、两端斜率为零），错距由
    净距需求反解最小值；环沿鼻端中心线的位置由 LLM 反解——闭合束
    中心线总长严格等于专利平均匝长，姿态不参与凑长。`rd2` 圆角在
    环端切点 Q 与卷环 G1 相切；无解参数会明确拒绝，不会延长成回头
    路径。非接线端整环保持同一匝位；接线端沿卷环以同心螺旋光顺
    过渡到相邻匝位。
    每股自身绝缘 + **匝绝缘分层**（每一匝、每一层均由封闭实体组成；
    卷环/换匝密站位放样的云母带拆为“主体段 + 四条卷环条带”，避免
    CAD 将有孔光顺放样误判为曲面体）
    + **对地绝缘方壳分层**包整束 + 可选**槽部防晕层**和**槽内固定件**；
    两根**引线**从同一接线端的槽底极限匝引出，先用两段相反的等角
    圆弧向槽底方向最小错位，再恢复为轴向伸出（自由直段长度=ysc，
    两段圆弧半径=引线折弯半径），
    端头可留**裸铜**。“出线端位置”可选 `+Z`（端部侧视图右端）或 `-Z`
    （左端），切换时两根引线作为整体镜像；`config.txt` 键为 `出线端在正轴端`；
  - **简化束模型**：铜导体等效整束 + 对地绝缘分层壳（生成快、文件小）。

## 快速上手

> 详细教程（参数含义、结果解读、警告处理、FAQ）见 **[docs/使用教程.md](docs/使用教程.md)**。

依赖用 [uv](https://docs.astral.sh/uv/) 管理，无需自装 Python：

```powershell
cd CoilDrawing
uv sync          # 1. 首次安装依赖（仅第一次）
uv run main.py   # 2. 启动界面
```

操作步骤：

1. **启动**后先选择（或新建）**工作空间目录**；窗口默认定位到上次目录，
   但每次都由用户确认。取消会直接退出；确认空目录后自动生成 `config.txt`
   与 `output\`。界面已预填专利算例参数（1180 内径 / 108 槽 / 节距 9 / 8 匝）；
2. **左侧表单**改成你的电机数据：铁芯与槽、绕组线规（支持两种线规并绕）、绝缘（均按**单边厚度**填）、端部结构；
3. 三维分层绝缘在两张表中设置：**对地绝缘分层**（总厚应 ≈ T2）与**匝绝缘分层**（总厚应 ≈ T1）；
   当前三维实体要求 **T1=T3 且 T2=T4**；若槽内/端部厚度不同，参数计算与二维图仍可使用，STEP 会为避免错误实体而停止生成；
   "三维模型"组里还可选 逐匝精细/简化束、引线折弯半径、端头裸铜长、
   出线端位置（`+Z`/`-Z`，两根引线一起镜像）、防晕层及槽楔/垫片开关；
4. 按 **F5**（或点"计算"）重新计算——改参数后不会自动重算；参数自动写入工作空间 config.txt
   （也可直接编辑该文件，软件会提示载入）；
5. 右侧查看**计算结果**表（按专利步骤 1–6 分节，底部红色 ⚠ 行是警告，务必阅读）和**线圈大样图**页；
6. 左下角按钮导出（默认到 `工作空间\output\`）：**Excel**（输入/结果/警告分表）、**CSV**、
   **大样图（PNG/PDF/SVG）**、**STEP 三维模型**（逐匝精细约 15~60 秒，简化束数秒）；
7. 需要 Parasolid 时：用 SolidWorks / NX / Solid Edge 打开 STEP → 另存为 `.x_t`（无损）。

运行测试（专利算例验证）：

```powershell
uv run pytest
```

## 打包成免安装 exe（Windows 10/11）

一键打包（PyInstaller 由 uv 管理，无需手工装包）：

```powershell
powershell -ExecutionPolicy Bypass -File tools\build_exe.ps1
```

产物在 `dist\CoilDrawing\`：**整个文件夹就是软件**，拷到任意 Win10/11 x64 电脑，
双击 `CoilDrawing.exe` 即用（不需要 Python、uv 或联网）。分发时把文件夹打成 zip 即可。

打包后可用开发/交付验收用的**自检模式**验证（生成大样图 + STEP 并退出，
全部成功退出码为 0）：

```powershell
dist\CoilDrawing\CoilDrawing.exe --smoke 自检输出目录
```

自检模式不是普通用户的工作模式。它会在指定测试目录中自动检查 Qt 界面、
计算引擎、大样图和 STEP 导出，并有意跳过启动时的工作空间选择窗口，避免
自动化验证被弹窗阻塞；不会使用或改写上次的正常工作空间。

说明：采用文件夹形式而非单文件 exe，是因为软件含 OCCT 几何内核（数百 MB），
单文件模式每次启动都要解压到临时目录，会慢很多。首次运行若被 Windows
SmartScreen 拦截（未签名程序的正常提示），点"仍要运行"即可。

## 关于 Parasolid (.x_t)

Parasolid 是西门子的**私有**几何内核格式，开源工具链无法直接写出合法的 .x_t 文件。
本工具导出的 **STEP (AP214)** 是无损 B-Rep 实体，转换路径：

1. 用 **SolidWorks / NX / Solid Edge**（它们本身就是 Parasolid 内核）打开导出的 `.step`；
2. 另存为 → Parasolid `.x_t`，即可得到零损失的 Parasolid 模型。

如需程序直接输出 .x_t，需要接入商业 SDK（如 CAD Exchanger Python API），
`coildrawing/model3d.py` 的导出层已按可替换设计。

## 输入参数对照（专利符号）

| 分组 | 参数 |
| --- | --- |
| 铁芯与槽 | D2 内径、LC 铁芯长、NS 槽数、TAW 节距、HS/WS 槽深宽、HSD 槽楔、WIHU/WIHM/WIHB 垫片 |
| 绕组线规 | N 匝数；导线1/2 的裸线宽厚 WB/WT、自身绝缘 T0、并绕根数 NPD、每匝层数 NCD |
| 绝缘(单边) | T1/T3 匝间（槽内/端部）、T2/T4 对地（槽内/端部）、CS 防晕（兼作 3D 防晕层厚度）；当前 3D 要求 T1=T3、T2=T4 |
| 端部结构 | LD 齿压板、LE 伸出长、F 鼻端抬高、seita3（度，配置键 `seita3_deg`）、RD 鼻端内弯半径（圆冠中心线路径半径 `Rc=RD+WA/2`）、RD1/RD2、rd1/rd2 弯曲半径、Ba 端部间隙、ysc 引线长、ξ 迭代误差；直鼻臂长度 Larm 由 LLM 守恒自动反算 |
| 3D 对地/匝绝缘分层 | 每层名称 + 单边厚度（对地总厚应≈T2、匝绝缘总厚应≈T1，不一致时给出警告） |
| 3D 模型选项 | 逐匝精细/简化束、引线折弯半径、端头裸铜长、出线端位置（`+Z`/`-Z`；配置键 `出线端在正轴端`）、防晕层（开关/每端伸出长度）、槽楔/槽楔下垫片/层间垫片/槽底垫片开关 |

## 与专利算例的验证

`tests/test_engine.py` 用专利具体实施方式 [0136]–[0237] 的算例做回归：
截面链路（WA/HAD/W/H/HD/WD/WC/HC）、几何链路（RR1/RR2/fai/t1/t2/gf/gi/弯弧判定）
按算例值精确断言；端部链路（CC=92、seita1=0.7112、S1=141、X1=1298.7、K1=54.97 等）
按 ±1% 断言。专利算例正文存在少量自相矛盾之处（如 D2=1180 与 RR1=467 不符、
h_=79 与其公式不符），代码一律以权利要求公式为准，详见测试文件注释。

## 目录结构

```
src/coildrawing/
  engine.py     计算引擎（专利步骤1-6 + 股线排布）
  drawing2d.py  大样图绘制（工程制图风格四视图）
  model3d.py    三维建模与 STEP 导出（逐匝精细/简化束）
  step_i18n.py  STEP 中文名称 \X2\ 转义修复
  export.py     Excel/CSV 导出
  config_io.py  工作空间 config.txt 读写（INI 风格带注释）
  ui.py         PySide6 界面（工作空间/菜单/自由文本数值框）
  compat.py     build123d 字体兼容导入
tests/          专利算例回归测试 + 配置往返 + 3D 实体有效性
tools/          开发辅助脚本（build_exe.ps1 打包、PDF 提取、3D 冒烟/渲染检查）
CoilDrawing.spec    PyInstaller 打包配置
docs/使用教程.md      详细使用教程（安装、参数、结果解读、FAQ）
docs/config_template.txt  默认参数模板（工作空间 config.txt 的样例）
docs/images/        教程配图（界面截图等）
docs/patent_pages/  专利扫描页（公式核对用）
```
